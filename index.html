04½ What it integrates

Editorial work has multiple shapes.
Mycroft brings the tools in.

Goose hosts the runtime; Mycroft wires in the editorial extensions. Day-to-day monitoring, deep-dive investigation, source archives — each handled by a purpose-built piece, all in one local workflow.

MONITORING

coJournalist.

Continuous awareness of your beat. Pulls feeds, flags signal, drafts the morning brief — so you walk in with leads, not noise.

INVESTIGATION

Spotlight.

Deep-dive on a name, a company, a place. Spotlight runs structured discovery across your sources and the open web — with citations, no hallucinated quotes.

COMING · MAY 2026

DataHound + Atelier.

DataHound — public-data discovery (registries, archives, leaks). Atelier — visual production for the published piece. Two more pieces shipping with the next release.

05 Install · five minutes

Form, script, run.
You're working.

No terminal-wrangling. The setup page generates a single script with your config baked in. One file, one run.

~/mycroft · setup

i. Fill in the setup form — cloud or local · API keys · plugins. Everything stays in your browser.

$ open https://mycroft.app/setup

your config never leaves the browser

ii. Download .command — a bash script with your selections embedded. Review it.

$ curl -O mycroft-setup.command

204 KB · plain text · nothing hidden

iii. Run it — double-click. Brew installs Goose + Obsidian, clones the pack, configures your shell.

$ chmod +x ./mycroft-setup.command && ./mycroft-setup.command

brew installs goose + obsidian · clones the pack · configures shell

✓ goose runtime · installed

✓ qwen 3.5 weights · loaded (4.2 GB)

✓ obsidian vault · ~/Documents/Mycroft

✓ plugins enabled · Spotlight · coJournalist

✓ local-first mode · ON

✓ zero data retention · confirmed

✓ Mycroft is ready — open a new terminal to begin.
