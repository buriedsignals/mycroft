Local model (sovereign mode)

🤗 Mycroft Qwen — abliterated journalism fine-tune local
Fine-tuned Qwen 3.5 on investigative methodology (SIFT, OSINT tool selection, evidence grounding, ethics). Abliterated — won't soft-refuse legitimate journalism queries. Two sizes:
  • 9B (default) — runs on 16 GB+ unified memory / RAM · 🤗 qwen3.5-9b-abliterated-journalist-GGUF
  • 35B (power) — needs 64 GB+ unified memory · 🤗 qwen3.5-35b-abliterated-journalist-GGUF (training in progress)
Runtime: llama-server on port 8080 (cross-platform GGUF). Apple Silicon users get native-speed inference from llama.cpp too; an MLX build is optional (set up separately if you want it).

Cost: $0 / month (hardware + electricity only)

Download + start (after your setup script runs): 
# 1. Install the HF CLI if you don't have it
pip install -U "huggingface_hub[cli]"

# 2. Download the 9B (pick a quantization — Q4_K_M is a good balance)
hf download tomvaillant/qwen3.5-9b-abliterated-journalist-GGUF \
  --local-dir ~/.mycroft/models/qwen-journalist-9b \
  --include "*Q4_K_M.gguf"

# 3. Start llama-server
llama-server -m ~/.mycroft/models/qwen-journalist-9b/*.Q4_K_M.gguf \
  -c 32768 --port 8080
Goose will then route to it when sovereign mode is on. 35B build is in progress — same flow once released.

Required prerequisites

Goose required
The agent runtime Mycroft plugs into. Desktop app (recommended for non-terminal users) or CLI. Download manually
Manual install (no Homebrew)
  1. Desktop: Download from the link above, unzip, run.
  2. CLI: curl -fsSL https://github.com/aaif-goose/goose/releases/download/stable/download_cli.sh | bash
  3. Verify: goose --version.
Obsidian Desktop required
Mycroft reads/writes your knowledge vault through Obsidian. Obsidian 1.12+ ships with a built-in CLI (obsidian-cli) — both the app and CLI come with a single Homebrew install. Download manually
Two manual steps (Obsidian 1.12+):
  1. After Obsidian launches, go to Settings → General → Advanced → Command Line Interface and toggle it ON. This can't be automated; Obsidian exposes it only through the app UI.
  2. Leave Obsidian Desktop running whenever you invoke morning-brief or vault-sync — the CLI talks to the live app. Writes silently fail if it's closed.
Firecrawl CLI required
Powers every web-capable recipe. The only permitted web-fetch tool in Mycroft. Get API key at firecrawl.dev
