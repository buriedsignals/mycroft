01 Sovereignty

Where the work happens.

Pick one below. Cloud is fast and cheap — ZDR open-weight providers in the US/EU. Local is yours, fully — a fine-tuned Qwen on your machine. Whichever you choose is the default; the other side stays available as fallback.

02 Local model

Mycroft Qwen.

Fine-tuned Qwen 3.5 on investigative methodology — SIFT, OSINT tool selection, evidence grounding, ethics. Abliterated: won't soft-refuse legitimate journalism queries.

Browser-only hardware probe. No data leaves this page.
-Hardware recommendation

system_profiler SPHardwareDataType | grep -E 'Memory|Chip|Cores'
Download + start (after your setup script runs): 
# 1. Install the HF CLI
pip install -U "huggingface_hub[cli]"

# 2. Pull the 9B (Q4_K_M is a good balance)
hf download tomvaillant/qwen3.5-9b-abliterated-journalist-GGUF \
  --local-dir ~/.mycroft/models/qwen-journalist-9b \
  --include "*Q4_K_M.gguf"

# 3. Start llama-server
llama-server -m ~/.mycroft/models/qwen-journalist-9b/*.Q4_K_M.gguf -c 32768 --port 8080

07 The script

Review.
Then run.

The exact bash that will land in your downloads folder. Your keys are baked in plaintext — review before running. It writes secrets to ~/.mycroft/.env, Goose config to ~/.config/goose/, and an updater job for repo updates.

Downloaded. Extract the ZIP, move mycroft-setup.command to the folder where you keep installers, then run it: 
chmod +x /path/to/mycroft-setup.command
/path/to/mycroft-setup.command
On macOS, right-click mycroft-setup.command → Open → Confirm the first time. After that, plain double-click works. The agent setup ZIP is separate and contains no secrets.
